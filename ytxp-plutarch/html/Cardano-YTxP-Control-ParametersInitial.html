data ControlParametersInitial Source #

Parameters available to the YieldListValidator and YieldListMP during compilation (therefore not containing any script hashes).

This is a GADT because the nonces must be serializable (and thus haskell types) as well as able to be applied to plutarch scripts (and thus PTypes).

To load the scriptToWrap arguments, see unsafeTermFromScript from Cardano.YTxP.Control.Utils
