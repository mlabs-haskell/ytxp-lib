getYieldedToHash :: YieldListSTCS -> Term s (PBuiltinList PTxInInfo :--> (PYieldingRedeemer :--> PYieldedToHash)) Source #

Given a list of reference inputs and a Yielding Redeemer, dig out the YieldList by: - Indexing the reference inputs according to the redeemer - Checking the fetched reference input for the correct YieldListSTCS - Decoding its datum (unsafely; the presence of the YieldListSTCS ensure it is authentic and well-formed) - Looking in the datum at the index in the redeemer and returning the YieldedToHash
